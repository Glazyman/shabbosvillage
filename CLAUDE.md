# Project Brain Rule

After every session in which any file is changed, added, or deleted — append a new entry to `_brain/PROJECT_BRAIN.md` under the `## Session Log` section. Format:

```
### YYYY-MM-DD — <short title>

- What changed and why (bullet points)
```

Use today's actual date. One entry per session. Do this at the end of the session before stopping.
